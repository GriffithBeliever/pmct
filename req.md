# Personal Media Collection Tracker


## Minimum Features:

-  Media Management: Add, edit, and delete media items (movies, music, games with title, creator, release date, and other metadata).

- Status Tracking: Mark items as "owned," "wishlist," "currently using," or "completed."

- Search: Find media by title, creator, genre, or other fields.

- Implement any AI features you can think of.


## Requirements:

- Provide your source code and a brief README explaining how to run it.

- Deploy the app and provide a URL for live testing.

- Add user authentication with profiles to share collections.

- Add extra features as you see fit, to demonstrate your creativity.
